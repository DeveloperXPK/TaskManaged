IsDoned

IsDoned es una aplicación web que te permite llevar un control de tus tareas pendientes y completadas.

Con IsDoned podrás:

  • Crear tareas
  • Marcar tareas como completadas
  • Eliminar tareas

¡Regístrate y comienza a organizar tus tareas!

¿Cómo funciona?

Para comenzar a utilizar IsDoned, sigue los siguientes pasos:

  1. Regístrate en la aplicación
  2. Inicia sesión
  3. Crea una tarea
  4. Marca la tarea como completada
  5. Elimina la tarea

¡Así de fácil!
